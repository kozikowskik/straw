# Security Policy

## Supported Versions

`straw` is currently pre-release v0 software. Security fixes are handled on the latest code line until a broader support policy is published.

## Reporting A Vulnerability

Please report security concerns privately through GitHub private vulnerability reporting:

https://github.com/kozikowskik/straw/security/advisories/new

Do not open a public issue for suspected vulnerabilities. Include the affected version or commit, a description of the impact, and steps to reproduce when possible.
